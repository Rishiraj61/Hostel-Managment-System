Hostel Management System

This project is a console-based Hostel Management System in C++ designed to help administrators and students manage hostel operations efficiently. The system stores room and student data using file handling, allowing records to be saved and loaded across sessions.

Features
Administrator

Add new rooms with details such as seating capacity, gender type, and rent

Check in students and automatically assign suitable rooms

View available rooms

Search for registered students

Check out students from rooms

Display complete room and student summaries

Make announcements for students

Student

Apply for mess facility and calculate total fee

View roommates and room details

File complaints

Check weekly meal schedule

Reject mess facility

View announcements

Technical Details

Uses classes for Room, Customer, and Hostel Management

Supports up to 100 rooms and multiple students per room

Stores data in text files for persistent storage

This project serves as a complete and practical example of file management, data structures, and object-oriented programming in C++.
